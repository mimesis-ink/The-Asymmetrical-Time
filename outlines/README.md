# 大纲 (Outlines)

此目录用于存放各部作品的大纲和情节规划。

## 文件说明 (Files)

### book1-outline.md
第一部《不对称时间》的完整大纲

### book2-outline.md
第二部（续作）的大纲规划

### series-overview.md
整个系列的总体规划和方向

## 使用建议 (Usage Tips)

1. 在开始写作前，先完善大纲
2. 大纲可以随着创作过程调整，但要保持记录
3. 使用大纲跟踪情节线索和伏笔

## 大纲结构建议 (Outline Structure Suggestions)

- **三幕结构：** 开端、发展、高潮、结局
- **章节概要：** 每章的主要内容
- **情节线索：** 主线和支线的发展
- **角色弧光：** 角色在故事中的成长轨迹
