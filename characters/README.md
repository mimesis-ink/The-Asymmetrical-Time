# 角色档案 (Character Profiles)

此目录用于记录所有角色的详细信息，以保持故事的连贯性。

## 角色分类 (Character Categories)

### 主要角色 (Main Characters)

在 `main/` 子目录中创建主要角色的档案。

### 次要角色 (Supporting Characters)

在 `supporting/` 子目录中创建次要角色的档案。

### 其他角色 (Minor Characters)

在 `minor/` 子目录中创建其他角色的简要信息。

## 使用模板 (Using the Template)

复制 `character-template.md` 并根据需要填写角色信息。

## 命名规范 (Naming Convention)

使用格式：`[角色名].md`，例如：`张三.md`
