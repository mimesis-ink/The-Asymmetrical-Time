# 世界设定 (Worldbuilding)

此目录用于记录故事世界的各种设定，确保故事的一致性和连贯性。

## 内容分类 (Categories)

### 地理设定 (Geography)

在 `geography.md` 中记录：
- 地点描述
- 地图（如有）
- 重要位置

### 历史背景 (History)

在 `history.md` 中记录：
- 世界历史
- 重要事件时间线
- 历史对现在的影响

### 社会结构 (Society)

在 `society.md` 中记录：
- 社会制度
- 文化习俗
- 组织机构

### 技术/魔法系统 (Technology/Magic System)

在 `systems.md` 中记录：
- 技术水平
- 特殊能力或魔法系统
- 规则和限制

### 时间设定 (Timeline)

在 `timeline.md` 中记录：
- 故事年表
- 重要事件的时间顺序

## 注意事项 (Notes)

保持设定的内部一致性，避免前后矛盾。
